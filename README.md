# YOLO_mediapipe
# MediaPipe & YOLO Computer Vision Projects

## Overview

This repository contains four real-time Computer Vision projects developed using Python, OpenCV, MediaPipe, and YOLOv8. These projects demonstrate practical applications of Artificial Intelligence and Computer Vision in gesture recognition, driver safety, object detection, and helmet compliance monitoring.

---

## Technologies Used

* Python
* OpenCV
* MediaPipe
* YOLOv8 (Ultralytics)
* NumPy
* Pycaw
* Comtypes
* SciPy

---

# Task 1: Hand Gesture Volume Controller

### Description

A real-time hand gesture recognition system that controls the system volume using finger movements detected through a webcam.

### Features

* Real-time hand tracking
* Thumb and index finger distance measurement
* Dynamic volume adjustment
* Volume percentage display
* MediaPipe Hands integration

### Technologies

* OpenCV
* MediaPipe
* Pycaw

---

# Task 2: Driver Drowsiness Detection

### Description

A driver monitoring system that detects drowsiness by analyzing eye movements and eye closure duration using facial landmarks.

### Features

* Real-time face detection
* Eye Aspect Ratio (EAR) calculation
* Drowsiness alert generation
* Live webcam monitoring
* Driver safety assistance

### Technologies

* OpenCV
* MediaPipe Face Mesh
* SciPy

---

# Task 3: Object Detection and Counting

### Description

A real-time object detection system using YOLOv8 that identifies and counts objects appearing in a video stream.

### Features

* Object detection using YOLOv8
* Real-time video processing
* Object counting
* Bounding box visualization
* Detection statistics display

### Technologies

* OpenCV
* YOLOv8
* Ultralytics

---

# Task 4: Helmet Detection System

### Description

A safety monitoring system that detects whether riders are wearing helmets and generates statistics from the processed video.

### Features

* Helmet detection using YOLOv8
* Wearer and non-wearer classification
* Video-based analysis
* Detection statistics generation
* Safety compliance monitoring

### Technologies

* OpenCV
* YOLOv8
* Ultralytics
